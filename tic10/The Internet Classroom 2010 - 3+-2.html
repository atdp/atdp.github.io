4.1

Sorry, Mr. Powazek

Properly format the three paragraphs of text you created in Quest 2.1 with HTML.
Use header tags for the titles of each paragraph.
Separate paragraphs using horizontal rules.

16 XP

4.2

Unemployed

Create a resume for a make-believe person that is formatted properly with HTML.
Make up fictitious names/skills/experience to fill it out.
See this example if you aren't sure what sort of information goes on a resume.

16 XP

4.3

Om nom nom

Create a fictional restaurant menu that is formatted properly with HTML.
Create at least two separate sections (appetizers and entrées, for example), and at least three items per section.
Include prices and a menu title.

16 XP

HTML: Hyperlinks

5.1

A House is not a Home

Take the model home you made in Quest 3.2 and create an HTML page for each room.
Write a brief description on each page and put it in its proper folder.
Create hyperlinks that link the pages of rooms that connect to each other.

26 XP

5.2

Labyrinth

Make a maze of interconnected HTML pages using hyperlinks.
Plan by drawing a map of your maze before you begin creating pages.
Identify each node using titles and text, including a start and an end node.
Your pages must be contained within at least two separate directories.
Include at least two pages that link to at least three other pages.
Your finished maze must match your original map, and must have a solution.

26 XP

HTML: Tables

6.1

Survey says...

Create a short survey of simple questions to ask your group mates and present the data in a table.
Include at least three questions, and display them on your page with any variety of HTML list element.
Put the names of your group mates on one axis, and the question asked on the other.
Fill the table by putting each survey response in the correct cell.

40 XP

6.2

Thirty Days Hath

Create a calendar for the six weeks of ATDP.
Include day numbers in each cell, with days of the week along the top.
Include at least two descriptive dates (exam dates, birthdays, holidays, etc.).

40 XP

6.3

Bird's Eye View

Use a large table to create a text map of the webpages you created in Quest 5.1 or 5.2.
Label cells with the name of the room or maze node.
Make sure each room or maze node is surrounded by empty cells.
Place a line-looking character (e.g. hyphens, slashes, vertical line) between the cells of rooms or nodes that are connected.

10 credits
