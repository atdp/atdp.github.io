To make an unvisited link green, you would use:

a:link { color: green; }

To make a visited link blue, you would use:

a:visited { color: blue; }

To make a link turn red when you hover over it, you would use:

a:hover { color: red; }

To make a link turn yellow when you click on it, you would use:

a:active { color: yellow; }
