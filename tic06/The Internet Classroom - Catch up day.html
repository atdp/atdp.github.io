CSS Exercise

Make a webpage for the class syllabus. Like the "Home Page Goals" exercise, all you need to do is copy/paste the information into your own HTML document and mark it up. Use CSS to modify the document's appearance. It doesn't have to look exactly like it does on the TIC site, but you should try to follow the basic rules. (i.e. use a table for the basic course info, use a list for the grading section, etc.)

Feel free to add color if you like, since the current syllabus is pretty plain, and upload to your FTP and link via your index as always.

  • How does the Internet penetration of your region compare to others? Why might that be the case?
