If you want to embedd the code, just copy and paste your css code within the <head> tag. But you would have to copy and paste the long code into every html document

If you want to externally link your css to the html, which im assumin you want to cause this would mean you would only have to post one itsy bitsy link. This is the code:

<link rel="stylesheet" href="filename.css" type="text/css" media="screen" />

filename would be the name of your css file.
