2006/07/22 19:07:07 PDT by Forward Biased

For those of you watching at home, the problem is that CSS IDs cannot start with a number. IE and Safari are both loose, so both will render it correctly. Firefox is more strict, so it will ignore IDs like that.

The solution is to use IDs that start with letters, e.g. #a1, #a2, #blahblah1335, etc. (the JavaScript and CSS are just fine).
