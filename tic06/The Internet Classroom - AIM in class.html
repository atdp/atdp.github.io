2006/06/28 13:35:05 PDT by aqhong
Edited at 2006/06/28 13:36:56 PDT

Haha, okay.

Good things about it:

  • Completely free and open-source

  • No installation necessary (which makes it portable)

  • Extensible via plugins (which gives you access to pretty much all the features you'd have to pay for elsewhere *cough*TrillianPro*cough*)

  • Not a bloated mess, unlike Trillian

  • A native Windows app, unlike Gaim

  • I don't think I have to compare it to AOL's client XP

Bad things about it:

  • OSCAR protocol plugin isn't complete, which means certain advanced AIM features are missing

Actually, I wouldn't even have noticed that downside if Kenton hadn't mentioned it a while ago, because I personally don't use a lot of the features OSCAR provides. But I thought I'd mention it anyway.

Basically, I like Miranda because it's very customizable, and can be as simple or as sophisticated as you want it be :)
