Flash

  • Motion Tweening

    • Create a keyframe on frame A, paint your object and select all of it to CONVERT TO SYMBOL (graphic).

    • Create a keyframe on frame B, down the timeline somewhere, and move your symbol to where you want it to go.

    • Right-click on any frame between A and B on the timeline, CREATE MOTION TWEEN.

  • Motion Tweening w/Animation guides

    • Right-click the layer on which the objects you want to move are and CREATE MOTION GUIDE.

    • Draw path on the guide layer (pencil tool)

    • Go to frame A and move object to beginning of path. Go to frame B and move object to end of path.

  • Shape Tweening

    • Create an object

    • Do NOT convert to symbol; if the object is already a symbol, right click and "Break Apart" all the way down, until it can't be broken down anymore.

    • Create a keyframe later on the timeline. Delete the old object and draw a new one. Again, make sure this object is NOT a symbol and is broken down entirely.

    • Click on a frame between A and B. Check in the Properties down below the stage for a "Tween: <dropdownmenu>" option and select SHAPE TWEEN

  • Buttons

    • Paint a button and CONVERT TO SYMBOL (button)

    • Double click the button to access alternative "settings" of the button (up/down, etc.) Create a keyframe on these other settings if you want to the button to look different on certain actions

    • Open up the Behaviors window

    • Select your button and add a behavior, such as "Go to URL" or "Go to frame and stop" or "Go to frame and play"... etc.
