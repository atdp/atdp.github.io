It looks like what you're doing is pasting ASCII art from a source that's not formatted as plain text, so when it's converted to plain text in the code blocks, the spacing between the characters is off, which is why it doesn't look good. Make sure the place you're getting your art from is formatted as plain text; plain text will usually be in a monospace (typewriter-ish) font like Courier (as shown in the code blocks here).
